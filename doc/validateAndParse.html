validateAndParse(elementId) → {boolean}

Entry function for parsing JSON. Does perform and delegate validation functions. If browser is not IE, then validation of required fields will be performed via built-in HTML5 validation. Else, a custom made validation via Bootstrap v3 using tooltips is triggered.
